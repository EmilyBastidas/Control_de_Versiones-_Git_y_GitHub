# Control de Versiones con Git y GitHub

## Descripción

Repositorio de estudio y práctica sobre Control de Versiones utilizando Git y GitHub.

Aquí recopilo apuntes, conceptos, comandos y ejercicios para comprender cómo gestionar cambios en proyectos, mantener un historial de versiones y colaborar con otros desarrolladores.

---

## Objetivos

- Comprender qué es el control de versiones.
- Aprender a utilizar Git desde la terminal.
- Gestionar repositorios locales y remotos.
- Trabajar con ramas (branches).
- Resolver conflictos de fusión (merge conflicts).
- Utilizar GitHub como plataforma de colaboración.
- Aplicar buenas prácticas en proyectos reales.
